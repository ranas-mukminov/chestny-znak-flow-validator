# chestny-znak-flow-validator

Инструмент контроля целостности цепочек маркировки «Честный ЗНАК» между производителем, оптом и розницей. Помогает найти «висящие» коды, неверные статусы, ошибки в партиях и несоответствия между учётом и данными системы маркировки.

## Что умеет
- Импортировать выгрузки из УТ/ERP/учётных систем (CSV/XLSX/XML) и синтетические ответы API.
- Анализировать цепочки и статусы кодов, строить графы движения.
- Формировать отчёты по проблемным кодам и готовить тексты претензий/запросов контрагентам.
- Предоставлять CLI и web-интерфейс на FastAPI для запуска проверок.

## Архитектура
Импортеры → ядро проверки → AI-слой отчётов → Web/CLI.

## Безопасность и законность
- Примеры данных синтетические, без реальных кодов и ПДн.
- Работа с реальными системами должна соблюдаться в рамках договора с оператором и законодательства РФ.
- Проект не автоматизирует отправку данных в ГИС МТ, а только анализирует выгрузки.

## Быстрый старт
1. Установите зависимости: `pip install -e .`
2. Выполните пример проверки: `python -m cz_validator.cli.main validate --producer examples/data/producer_export_sample.csv`.
3. Сгенерируйте отчёт: `python -m cz_validator.cli.main generate-report examples/reports/discrepancies_report_sample.md`.

## Услуги и поддержка – run-as-daemon.ru

Этот проект развивается инженером DevOps/DevSecOps с сайта [run-as-daemon.ru](https://run-as-daemon.ru).

Если вам нужно:
- навести порядок в маркировке и цепочках оборота;
- понять, где теряются коды между производителем, оптом и розницей;
- получить регулярные отчёты по качеству данных для руководителя,

вы можете заказать:
- аудит процессов маркировки и обмена кодами;
- внедрение `chestny-znak-flow-validator` под вашу 1С/ERP;
- настройку регулярной проверки и отчётности.

Проект независим от оператора «Честный ЗНАК», Банка России и госорганов; не заменяет юридическую экспертизу и не является официальным средством исполнения требований законодательства.
