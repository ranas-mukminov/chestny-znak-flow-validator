# chestny-znak-flow-validator

Open source toolkit for validating the integrity of labelled-goods flows across the Russian "Честный ЗНАК" supply chain (producer → wholesaler → retail). The repository follows strict TDD, linting, and security checks before any pull request is merged.

Key capabilities:
- Import synthetic CSV/XLSX/XML extracts from ERP/WMS/UT systems.
- Build simplified movement graphs and verify statuses/quantities.
- Produce human-friendly discrepancy reports and draft claim texts.
- Provide CLI and lightweight FastAPI web app for running validations.

See [README.ru.md](README.ru.md) for the detailed Russian description, legal notes, and onboarding guidance.
